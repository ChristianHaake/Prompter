Kurzbeschreibung

Ein browserbasierter Teleprompter für Präsentationen, Erklärvideos, Podcasts und Redebeiträge mit integrierter Zeitsteuerung.

Zielgruppe

* Schülerinnen und Schüler
* Lehrkräfte
* Fortbildende

Einsatzszenarien

* Präsentationen
* Erklärvideos
* Podcasts
* Nachrichtenformate
* Videostatements
* Sprachprüfungen

User Story

Als Lernende möchte ich meinen Sprechtext automatisch scrollen lassen, damit ich flüssig präsentieren und eine vorgegebene Zeit einhalten kann.

Akzeptanzkriterien

* Text kann erstellt oder eingefügt werden.
* Scrollgeschwindigkeit oder Zielzeit kann eingestellt werden.
* Vollbildmodus ist verfügbar.
* Projekt wird lokal gespeichert.
* Export/Import als JSON ist möglich.

MVP

* Texteditor
* Auto-Scroll
* Countdown
* Geschwindigkeitsregler
* Vollbild
* Local Storage

Erweiterungen
